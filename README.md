# Проект: Взаимодействие с LLM через API

Этот проект демонстрирует работу с языковыми моделями через API сервиса GigaChat.

## Общая настройка

### Установка зависимостей

Убедитесь, что установлены все необходимые зависимости:
```bash
pip install gigachat
```

### Настройка конфигурации

Настройте конфигурацию в файле `config.py`, указав следующие параметры:
- `GIGACHAT_API_KEY` — ваш API-ключ
- `GIGACHAT_SCOPE` — необходимая область доступа
- `GIGACHAT_MODEL` — модель, которую вы хотите использовать
- `CERTIFICATE_PATH` — путь к сертификату (если требуется)

## День 1: Первый запрос к LLM через API

Этот скрипт демонстрирует отправку запроса к модели языкового процессора (LLM) через API сервиса GigaChat и вывод ответа в консоль.

### Функционал

- Отправка пользовательского запроса к модели GigaChat
- Получение и вывод ответа модели
- Отображение статистики использования токенов:
  - Количество токенов в запросе (prompt tokens)
  - Количество сгенерированных токенов (completion tokens)
  - Общее количество использованных токенов
  - Количество кэшированных токенов

### Использование

1. Запустите скрипт:
   ```bash
   python day1.py
   ```

2. Введите ваш запрос в консоль и получите ответ от модели.

## Пример работы

```
Введите запрос для LLM GigaChat:    Расскажи о себе

Привет! Я — языковая модель, разработанная для помощи в ответах на вопросы, создании текстов и решении различных задач.

Prompt tokens: 15, completion token: 25, total tokens: 40, cached tokens: 0
```

## День 2: Контроль формата ответа

Этот скрипт демонстрирует различные способы контроля ответа от LLM через API GigaChat.

### Функционал

- Сравнение ответов на один и тот же запрос:
  - Без ограничений
  - С явным описанием формата ответа
  - С ограничением по длине (максимальное количество токенов)
  - С условием завершения ответа (stop sequence)
- Использование system prompt для управления структурой ответа
- Тестирование различных методов ограничения вывода модели

### Использование

1. Убедитесь, что конфигурация в `config.py` уже настроена

2. Запустите скрипт:
   ```bash
   python day2.py
   ```

3. Программа автоматически выведет и сравнит четыре различных типа ответов.

### Пример работы

```
Ответ по умолчанию:
-------------------
Астрономия — это наука о Вселенной, её объектах и явлениях...

Ответ в соответствии с описанным форматом:
------------------------------------------
Астрономия — это наука, изучающая небесные тела и космические явления.

Астрономия необходима для понимания структуры Вселенной, происхождения и эволюции звёзд, планет и других космических объектов.

Ограниченный по длине ответ:
----------------------------
Астрономия — это наука о

Условие завершения ответа:
--------------------------
Астрономия — это наука о Вселенной
```

## Требования

- Python 3.8+
- Библиотека `gigachat`
- Действующий доступ к API GigaChat
